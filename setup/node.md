## Permissão

Para dar permissão total de escrita (leitura, escrita e execução) a todos os usuários em todos os diretórios e arquivos dentro da pasta do projeto teste-api-e2e, execute o seguinte comando no terminal, dentro da pasta do projeto:

```bash
chmod -R 777 .
```
